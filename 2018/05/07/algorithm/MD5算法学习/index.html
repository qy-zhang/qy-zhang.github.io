MD5算法学习

Posted on 2018-05-07 | In Note

转载一篇博客学习MD5算法。

MD5介绍

MD5 即 message-digest algorithm 5（信息-摘要算法），它只是一个摘要算法，并不是加密算法。

加密后的消息是完整的，具有解密算法，能够得到原始数据；摘要得到的消息是不完整的，通过摘要的结果，不能得到原始数据。

MD5是不可逆的：没有对应的算法，从生产的md5值逆向得到原始数据。
MD5不具有唯一性：一个原始数据，只对应一个md5值；但是一个md5值，可能对应多个原始数据。所以无法将MD5作为数据库的主键。

MD5表现形式

md5的长度，默认为128bit，也就是128个0和1的二进制串。
二进制不方便表示，所以被换成16进制表示。每4个bit表示一个16进制，128/4 = 32，所以在16进制表示下，md5为32位。
16位的md5是将32位md5去掉前八位，去掉后八位得到的。

MD5作用

  • 一致性检验，比如很多网站提供下载文件的md5值。
  • 数字签名，还是检验问题。只是把md5看成一个指纹，作为独一无二的标识。
  • 安全访问认证，在用户注册时，会将密码进行md5加密，存到数据库中，这样就不会明文传输用户密码了。
