Vim常用命令记录

Posted on 2018-05-29 | In Note

Vim中的一些常用命令记录

浏览操作

移动光标

按键 作用
space 光标右移一个字符
Backspace 光标左移一个字符
Enter 光标下移一行
nG 光标移至第n行首
n+ 光标下移n行
n- 光标上移n行
ctrl+u 向上翻半屏
ctrl+f 向上翻一屏
ctrl+d 向下翻半屏
ctrl+b 向下翻一屏

标记

创建标记

将光标移到某一行，使用ma命令进行标记。
可以使用小写字母a-z或大写字母A-Z中的任意一个做为标记名称。小写字母的标记，仅用于当前缓冲区；而大写字母的标记，则可以跨越不同的缓冲区。例如，你正在编辑File1，但仍然可以使用’A命令，移动到File2中创建的标记A。

利用:marks命令，可以列出所有标记。这其中也包括一些系统内置的特殊标记（Special marks）：
| 标记 | 作用 |
| - | - |
| . | 最近编辑的位置 |
| 0-9 | 最近使用的文件 |
| ∧ | 最近插入的位置 |
| ‘ | 上一次跳转前的位置 |
| “ | 上一次退出文件时的位置 |
| [ | 上一次修改的开始处 |
| ] | 上一次修改的结尾处 |

跳转标记

创建标记后，可以使用'a命令，移动到指定标记行的首个非空字符。这里’是单引号。也可以使用a命令，移到所做标记时的光标位置。这里是反引号（也就是数字键1左边的那一个）。

退出

按键 作用
ZZ 保存当前文件并关闭，最后一个文件退出
wq 将缓冲区内的资料写入磁盘中并退出，同:x
!q 丢弃所有的修改并退出
!e 放弃所有修改并重新载入该文件的原始内容
:only 关闭除当前窗口外的所有其它窗口，如果这些窗口中有被修改过的会得到一个错误信息，同时那个窗口会被留下来
:close 关闭窗口，需要先保存修改，不会关闭最后一个窗口
“quit 会关闭最后一个窗口

编辑操作

选定文本块

按键 作用
v 从光标当前位置开始，光标所经过的地方会被选中，再按一下v结束
V 从光标当前行开始，光标经过的行都会被选中，再按一下V结束
ctrl+v 块选择，从光标当前位置开始，选中光标起点和终点所构成的矩形区域，再按一下ctrl+v结束
ggVG 全选，gg为跳到行首，V选中整行，G末尾

复制粘贴

复制

复制的命令是y，即yank（提起）
| 按键 | 作用 |
| - | - |
| y | 在使用v模式选定了某一块的时候，复制选定块到缓冲区 |
| yy | 复制整行（nyy或者yny ，复制n行，n为数字） |
| y^ | 复制当前到行头的内容 |
| y$ | 复制当前到行尾的内容 |
| yw | 复制一个word （nyw或者ynw，复制n个word，n为数字） |
| yG | 复制至档尾 |

粘贴

粘贴的命令式p，即put（放下）
| 按键 | 作用 |
| - | - |
| p | 在当前位置后粘贴（如果是整行的话粘贴到下一行） |
| P | 在当前位置前粘贴（如果是整行的话粘贴到上一行） |

寄存器

Vim 中的复制、删除的内容都会被存放到默认（未命名）寄存器中，之后可以通过粘贴操作读取默认寄存器中的内容。

寄存器是完成这一过程的中转站，Vim 支持的寄存器非常多，其中常用的有 a-zA-Z0-9+”% 等，下面介绍各个寄存器的作用
| 寄存器名 | 作用 |
| - | - |
| “（单个双引号） | 未命名的寄存器，是 Vim 的默认寄存器，缓存最后一次操作内容 |
| 0-9 | 数字寄存器，其中0寄存器保存最近一次的操作内容，1-9寄存器保存最近九次操作内容 |
| a-zA-Z | 用户寄存器，小写覆盖，大写追加 |
| +（加号） | 剪切板寄存器，关联系统剪切板，保存在这个寄存器中的内容可以被系统其他程序访问，也可以通过这个寄存器访问其他程序保存到剪切板中的内容 |
| %（百分号） | 当前文件名 |
| #（井号） | 当前交替文件名 |
| :（冒号） | 上次执行的ex命令 |
| /（斜划线） | 上次查找的关键字 |
| -（横杠） | 黑洞寄存器，不缓存操作内容，"_dd删除当前行而不是剪切 |

（如果想了解更多寄存器内容，可参看）

通过reg或者dis命令可以查看所有寄存器中的内容，也可以只查看指定寄存器的内容（将寄存器名称作为参数）::reg [register_name]

下面通过几个例子介绍寄存器的使用，要使用寄存器就要在寄存器名字前面加上”（单个双引号），比如"a表示使用寄存器a。

在插入模式，当我们按下 + R键，再加上相应的寄存器的名字，就可以插入寄存器中的内容了。

从Vim中复制到Vim之外

通过上面寄存器的介绍，”（单个双引号）这个寄存器是 Vim 默认寄存器，+（加号）这个寄存器是与系统剪切板关联的地方，如果需要将 Vim 剪切板中的内容粘贴到 Vim 之外的地方，先将内容复制到+（加号）寄存器中即可。

剪切

剪切的命令是d，即delete，d与y命令基本类似，所以两个命令用法一样，不加赘述。
| 按键 | 作用 |
| - | - |
| d | 剪切选定块到缓冲区 |
| dd | 剪切整行 |
| d^ | 剪切至行首 |
| d$ | 剪切至行尾 |
| dw | 剪切一个word |
| dG | 剪切至档尾 |

删除与修改

按键 作用
x 删除光标所在字符，等同于Delete功能键
X 删除光标前字符，相当与Backspace
r 修改光标所在字符
R 替换当前字符及其后的字符，直到按ESC
s 删除光标所在字符，并进入输入模式直到ESC
S 删除光标所在整行，并进入输入模式直到ESC

撤销

按键 作用
u 撤销上一个编辑操作
ctrl+r 恢复，即回退前一个命令
U 行撤销，撤销所有在前一个编辑行上的操作

插入

按键 作用
a 在当前位置之后插入
A 在当前行尾插入
i 在当前位置之前插入
I 在当前行首插入
o 新增一行于当前行下方，并在新行插入
O 新增一行于当前行上方，并在新行插入
