snippet的开始几个参数设置

name : snippet name
snippet的名字，表现在YASnippet菜单上，起一个浅显易懂的名字很关键
key : snippet abbrev
snippet的触发键，当按下如上描述的key之后当前编写的snippet就会被触发
binding : direct keybinding
snippet的直接快捷键触发
