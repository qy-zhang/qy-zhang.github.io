字符串转数字

atoi

C语言中常用的atoi就是字符串转整形数字，能够将一个C风格的字符数组转换成整数。

atoi实现

实现过程就是从低到高将字符串的数字每一位乘10加过去，最后得到最终结果
同时要注意正负号，空格，数字越界之列的小问题

atoi使用

使用过程如下：由于atoi的函数参数是c风格的字符数组char[]，在C++中要使用c_str()来将string转换成这种风格

c_str()介绍

const char* c_str() const
作用是将string转换为const char*。反过来如果想要将const char*转换成string，直接用一个构造函数即可

stringstream

C++引入了ostringstream、istringstream、stringstream这三个类，要使用他们创建对象就必须包含sstream.h头文件。
istringstream类用于执行C++风格的串流的输入操作，从istream和stringstreambase派生而来；
ostringstream类用于执行C风格的串流的输出操作，从ostream和 stringstreambase派生而来；
stringstream类同时可以支持C风格的串流的输入输出操作，从iostream类和stringstreambase派生而来。

istringstream

istringstream是由一个string对象构造而来istringstream::istringstream(string str);，从一个string对象读取字符。
构造字符串流的时候，空格会成为字符串参数的内部分界。
str()成员函数的使用可以让istringstream对象返回一个string字符串。

ostringstream

ostringstream也是由一个string对象构造而来ostringstream::ostringstream(string str);，向一个string插入字符。
通过put()或者左移操作符可以不断向ostr插入单个字符或者是字符串，通过str()函数返回增长过后的完整字符串数据。
但是注意，如果构造的时候对象内已经存在字符串数据的时候，那么增长操作的时候不会从结尾开始增加，而是修改原有数，超出的部分增长。

stringstream

用于C++风格的字符串的输入输出的，构造函数stringstream::stringstream(string str);
