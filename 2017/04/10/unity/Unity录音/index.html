Unity录音

发表于 2017-04-10 | 分类于 Blog

项目中需要使用Unity录音，然后通过多音调检测算法来得出每一段音频演奏了哪些音符。

录音

Unity录音功能主要通过Microphone这个类来完成。Microphone
使用方法如下：

Microphone.Start函数声明如下：
public static AudioClip Start(string deviceName, bool loop, int lengthSec, int frequency);
loop表示如果录音时长lengthSec达到了之后，是否还继续录音，如果是的话录音结果重新从头开始存。

录音结束函数声明如下：
public static void End(string deviceName);
这个deviceName和开始录音的参数意义一样，如果传入的是null或者空字符串，默认microphone被执行相应函数。

录音得到的数据

通过Microphone.Start的函数声明可以看到录音的返回值为AudioClip类型。AudioClip继承自Object类，用来存放录音数据，基本所有需要的录音数据都被存放在其中。

录音采样点数samples

AudioClip类型有一个readonly的成员变量samples，用来记录当前录音片段的采样点数。
有了输入的采样率，在录音时我们对返回值取的一般是采样点数。当我们设置4096个采样点为一帧的时候，通过判断AudioClip.samples的数值就可以知道录音数据是否达到了一帧。

录音数据

  1. 获取录音数据
    AudioClip中有GetData成员函数来得到采样数据，函数声明如下：
    public bool GetData(float[] data, int offsetSamples);
    offsetSamples参数表示从第几个位置开始取。
    data表示存放取出的采样点，取出的采样点数为data数组的长度。采样数据是经过归一化的，值在-1到1之间。（归一化就是除以了32767）

  2. 设置录音数据
    public bool SetData(float[] data, int offsetSamples);
    分析同获取录音数据
