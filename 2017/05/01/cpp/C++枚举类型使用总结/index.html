C++枚举类型使用总结

发表于 2017-05-01 | 分类于 Note

枚举类型是C++中的一种派生数据类型，它是由用户定义的若干枚举常量的集合。

枚举类型介绍

枚举类型与其他的比较

我们写程序的时候，经常会需要将一些场景、对象关联成一组，作为可选项供不同的情况下挑选。比如说一周分为7天，天气分几种等。
典型的做法是定义三个常数，作为标志位来标志。比如说UNIX的三个流输入、输出、错误，就是用0、1、2来表示。这种做法比较麻烦，需要牢记每个标志位代表的含义。
枚举类型就可以解决这些问题

枚举类型的定义

枚举类型的定义格式为：enum <类型名> { <枚举常量表> };
枚举常量表中存放的是枚举成员，表示枚举类型的取值。枚举成员之间以逗号间隔，且必须各不相同。

枚举成员

枚举成员的取值

枚举成员代表了该枚举类型的变量能够取到的所有的值，编译系统为每个枚举成员指定一个整数值，在默认情况下，这个整数就是所列举元素的序号，序号从0开始。
enum week {Sun, Mon, Tue, Wed, Thu, Fri, Sat};，依次为0到6。
枚举成员还能够被显式指定值，我们可以指定Sun为1，那么后续依次累加，变为1到7了。
enum week {Sun=3, Mon, Tue=4, Wed, Thu, Fri, Sat}如果出现这样的形式，那么Sun为3，Mon累加为4，Tue被赋值还是为4，后续依次累加即可。

枚举成员取值范围

如果枚举中所有的枚举成员的值均为非负，该枚举的表示范围是0~2^k-1，其中2^k是能使所有枚举成员都能够位于此范围内的最小的2的幂；如果枚举成员的值有负数，那么就是 -2^k ~ 2^k-1

一个枚举类型的sizeof就是某个能够容纳其范围的整形的sizeof，而且不会大于sizeof(int)，在32位机器中，sizeof(int)一般等于4。
