Unity绘制五线谱

Posted on 2017-01-16 | In Blog

在Unity开发中，需要绘制乐谱。
使用乐谱字体 mscore-20.ttf ，使用该字体输入对应的unicode编码能够绘制出乐符。

Canvas

Canvas画布是摆放容纳所有的UI元素的区域。画布是一个游戏对象上的某个Canvas component组件，所有的UI元素必须是这个画布的子对象。

绘制乐谱使用的是UI组件Canvas(画布)，通过画布组件可以绘制我们需要的图形。

Text

每个乐符可以作为Canvas的一个Text(字体)组件
通过输入乐符字体mscore-20.ttf中对应的编码，来绘制对应的乐符。
Text组件中有很多参数，主要参数是设置字体内容Text.text，和字体font

可以通过代码根据需求绘制乐符

transform.SetParent

通过设置SetParent来将新建的对象放到相应的位置.
newNote.transform.SetParent(transform.FindChild("Canvas_Stave").FindChild("Notes"));
目录结构如下图所示，新建的四个Note放在了Canvas/Canvas_Stave/Notes目录下

transform.localScale

通过transform的localScale和localPosition来设置text的相对Canvas中的位置。
newNoteText.transform.localScale = new Vector3(1, 1, 1);

rectTransform.sizeDelta

通过sizeDelta设置text的大小(矩形框)。
如果下面设置居中对齐的话可以将这个地方的矩形框设置得大一点，能够将整个字体完整显示出来(如果设置的过小了的话会导致字体无法显示)
newNoteText.rectTransform.sizeDelta = new Vector2(300, 200);

alignment

由于是以乐符中心为锚点，要设置text组件的对齐方式为居中对齐
newNoteText.alignment = TextAnchor.MiddleCenter;

alignByGeometry

最后要注意一定要勾选 Align By Geometry 选项，在代码中体现为
newNoteText.alignByGeometry = true;
由于使用的是乐符字体，要设置这个属性为真
