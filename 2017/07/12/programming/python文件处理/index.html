Python文件处理

Posted on 2017-07-12 | In Blog

有关文件夹与文件的查找，删除等功能在os模块中实现。使用时需先导入这个模块：
import os
shutil包中也有一些文件处理功能。

目录操作

os包

获取当前目录os.getcwd()，返回字符串表示当前路径，相当于pwd
在当前目录下创建一个子目录os.mkdir(path)，相当于mkdir
创建子目录os.makedirs(path)，path为要创建的子目录的路径
删除子目录os.rmdir(path)，path为要删除的目录的路径，只能删除空目录，相当于rmdir

重命名os.rename(old, new)
更改当前目录os.chdir(path)，path为要切换到的目录的路径

更改文件目录的权限os.chmod(path, mode)，相当于chmod

判断一个路径（目录或文件）是否存在os.path.exits(path)，返回True/False表示path路径是否存在
判断一个文件是否存在os.path.isfile(path)，返回True/False表示path文件是否存在
判断一个文件是否存在os.path.isdir(path)，返回True/False表示path目录是否存在

获取目录中的文件及子目录的列表os.listdir(path)，返回一个列表，相当于ls
查看目录的附加信息os.stat(path)，相当于ls -l

切分路径和文件名os.path.split()，将路径和最后的文件名切分开来，返回一个列表，第1个表示路径，第二个表示文件名
获取路径名os.path.dirname()

切分扩展名os.path.splitext()
获取文件名（无后缀）os.path.basename()

获取文件大小os.path.getsize(path)

shutil包

要删除有内容的目录需要使用shutil.rmtree(path)，相当于rm -rf
复制文件到另一个目录shutil.copy(src, dst)，相当于cp
移动文件到另一个目录shutil.move(src, dst)，相当于mv

根据当前时间为名字来创建一个目录

获取指定目录下的所有子目录的列表

要把目录中子目录列出来，而不列出文件，所以判断isdir就行了

同理如果要列出文件而忽略文件夹，改为os.path.isfile(p+x)即可

os的一些其他操作

运行shell命令os.system()
终止当前进程os.exit()

读写文件

open(‘file’[,’mode’])函数打开文件，默认的mode是’r’读模式
read()函数读取文件内容，返回一个字符串，将换行符写成\n
readline([size])函数读取文件的一行信息，如果定义了size，就读取一行中的size大小部分
readlines([size])函数读出所有行，每一行作为list的一个成员，内部实现就是循环调用readline
write(string)函数将string字符串写入文件
writelines(list)函数将list中的每一个元素一行一行地写入文件
