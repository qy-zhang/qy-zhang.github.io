Linux开发常用命令

Posted on 2017-07-24 | In Note

进程相关

ps命令

ps(process status)命令用来列出系统中当前运行的进程，列出的是当前系统中进程的快照，就是执行ps命令的那个时刻的那些进程。
通过ps命令可以确认有哪些程序正在运行、运行的状态、进程是否结束等信息。

使用实例：
ps -A 显示所有进程信息
ps -u root 显示root用户进程信息
ps -ef 显示所有进程信息，包括命令行
ps aux 列出目前所有的正在内存中的程序

重点最常用的：
ps -ef | grep ssh 查找ssh进程，ps与grep组合用法

top命令

top命令用来实时显示系统中各个进程的资源占用情况，类似于windows的任务管理器。

kill命令

kill命令用来终止指定进程的运行。通常终止一个前台进程可以直接使用Ctrl+C，终止一个后台程序就需要使用kill命令了。

使用实例：
找到vim进程杀掉
先通过ps命令得到进程id号pid ps -ef | grep vim
kill pid

kill -9 pid 彻底杀死进程，-9表示信号，如果没有带这个信号号码选项，默认是发出15这个信号，表示终止

init进程是不可杀的，init进程是第一个进程，进程编号始终为1，其他所有的进程都是init进程的子孙。

killall命令相当于ps和kill的综合，直接杀死进程名，不需要进程id
vi & 后台挂载运行一个vi进程
killall -9 vi 直接杀掉所有vi进程

文件处理相关

cat命令

cat命令用来连接文件或标准输出并打印。常用来显示文件内容，或者将几个文件连接起来显示。

使用实例：
cat -n test1.txt test2.txt
将test1.txt和test2.txt文件内容加上行号后输入到控制台中。可以输出n个文件内容拼接在一起，不会影响原文件内容。

cat -n test1.txt > test2.txt
将test1.txt文件内容添加行号输入到test2.txt文件中，test2.txt文件内容被替换掉，控制台不输出信息。

cat > test.txt
创建一个文件，在控制台中输入文件内容，按下Ctrl+C来结束输入，输入的内容保存到文件中。

wc命令

wc(word count)命令用来统计指定文件中的字节数、字数、行数，并将统计结果输出显示。

使用实例：
假设有如下test.txt文件存在
hnlinux
peida.cnblogs.com
ubuntu
ubuntu linux
redhat
Redhat
linuxmint

wc test.txt 输出文件行数、字数、字节数和文件名，结果为 7 8 70 test.txt
wc -l test.txt 输出文件行数和文件名，结果为 7 test.txt
wc -c test.txt 输出文件字节数和文件名，结果为 70 test.txt
wc -w test.txt 输出文件字数和文件名，结果为 8 test.txt

通常wc命令会与ls、cat命令一起使用
cat test.txt | wc -l 输出文件行数
分析一下命名组成，首先通过cat命令将test.txt文件内容提取出来，然后通过wc命令来统计文件内容中的信息，所以最后只是输出文件的行数，没有文件，就不输出文件名了。

ls -l | wc -l 输出当前目录下的文件数

grep命令

grep(Global Regular Expression Print)命令是一种文本搜索工具，能够使用正则表达式搜索文本，并把匹配的行打印出来。
常用正则表达式：
a+表示a至少出现一次，a*表示a可以出现n次（0到n），a?表示a最多出现一次，a.表示a后面添加任意一个字符（除了换行）
[]表示一组字符，[^]表示不再[]中的字符，a{n}表示连续出现3个a

使用实例：
ps -ef | grep svn 查找指定进程，输出所有结果到控制台
ps -ef | grep -c svn 查找指定进程个数，输出个数到控制台
grep ‘linux’ test.txt 在test.txt文件中查找与linux匹配的行，输出。可以在后面添加多个文件
cat test.txt | grep -f test2.txt 在文件test.txt中匹配test2.txt中的所有行，输出匹配结果

find命令

find /home -name “*.txt” 在/home目录下查找以.txt结尾的文件名

linux下查找名字包含cvte的文件，再查找出的文件中找出包含abc的文件
find /home -name ‘cvte‘ | xargs grep “abc”

find命令把匹配到的文件传递给xargs命令

权限

Linux的文件权限可以分成三段，用户权限，组权限，其他权限。使用10个字符表示。
drwxr-xr-x
d表示是目录
第二到第四位表示文件所有者权限，读、写、执行
第五到第七位表示文件所有者所在的组权限，读、执行
第八到第十位表示其他用户权限，读、执行
r用4表示，w用2表示，x用1表示
所以上述drwxr-xr-x用数字表示就是755

一般使用chmod命令修改文件权限
另外还有一种比较方便的针对某一类的用户的操作：
先介绍2个操作符. “+” 表示增加权限. “-“ 表示消减权限。
文件所有者可以用字母u表示；用户所在的组可以用字母g来表示；其他人可以用字母o来表示；所有人可以用字母a来表示。（其实就是单词ser、group、other、all的第一个字母）

-R进行递归( recursive )的持续更改，即连同子目录下的所有文件、目录

awk

首先几下几个内置变量

| - | - |
| $0 | 当前记录（这个变量中存放着整个行的内容）|
| $1~$n| 当前记录的第n个字段，字段间由FS分隔|
| FS | 输入字段分隔符 默认是空格或Tab|
|NF| 当前记录中的字段个数，就是有多少列|
|NR |已经读出的记录数，就是行号，从1开始，如果有多个文件话，这个值也是不断累加中。|
|FNR| 当前记录数，与NR不同的是，这个值会是各个文件自己的行号|
|RS| 输入的记录分隔符， 默认为换行符|
|OFS| 输出字段分隔符， 默认也是空格|
|ORS| 输出的记录分隔符，默认为换行符|
|FILENAME| 当前输入文件的名字|

几个实例：
awk ‘$3==0 && $6==”LISTEN” ‘ netstat.txt #文件中第三列为0，第6列为LISTEN

深入学习

网络相关

ifconfig命令

ifconfig 显示已激活的网络设备信息
ifconfig eth0 192.168.1.1 前一个参数为具体网卡，后一个为配置的IP地址

netstat命令

netstat命令用来显示与IP、TCP、UDP和ICMP协议相关的统计数据，一般用于检验本机各端口的网络连接情况。

使用实例：
netstat -a 显示所有的有效连接信息，包括ESTABLISHED和LISTENING状态
netstat -nu 显示UDP连接状态，-n表示直接使用数字（IP地址）而不是使用域名，-u表示UDP

traceroute命令

traceroute命令用来追踪网络数据包的路由途径。
底层是应用ICMP协议和TTL
使用实例：
traceroute www.baidu.com

ping命令

ping命名用来测试与目标主机的连通性。
linux下ping不会自动终止，需要按Ctrl+C终止或者用参数-c指定要求完成的回应次数。

ping -c 10 192.168.120.206 指定ping次数

开发及调试

gcc

gcc/g++在执行编译工作的时候，总共需要4步

  1. 预处理，生成.i的文件，预处理器cpp.exe
  2. 将预处理后的文件不转换成汇编语言，生成文件.s，编译器egcs
  3. 有汇编变为目标代码(机器代码)生成.o的文件，汇编器as.exe
  4. 连接目标代码,生成可执行程序，连接器ld.exe

gcc选项主要有如下几种：

  • 总体选项
    -S激活预处理和编译，把文件编译成汇编代码
    -c激活预处理、编译和汇编，将文件编译成obj

  • 目录选项
    -ldir添加include目录
    -llibrary添加编译时使用的库

  • 调试选项
    -g产生调试信息

  • 错误与警告选项
    -w关闭警告
    -Wall显示所有警告

  • 优化选项
    -O0表示没有优化，-O1为默认值，-O3优化级别最高

gdb

启动gdb后，进入到交互模式，通过以下命令完成对程序的调试：
b设置断点；l查看源码；r运行程序；c继续执行；n表示step over；s表示step into；finish执行完当前函数；until直到退出循环体（后面可以加上行号）；q退出
p打印表达式；display单步时每次输出观察值；watch当监视的值改变暂停；whatis查询变量、函数
bt显示当前调用堆栈，up/down改变堆栈显示的深度；
