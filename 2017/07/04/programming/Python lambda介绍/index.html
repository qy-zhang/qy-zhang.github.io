Python lambda介绍

Posted on 2017-07-04 | In Blog

编程中提到的lambda表达式，通常是在需要一个函数，但是又不想费神去命名一个函数的场合下使用，也就是指匿名函数。

lambda介绍

匿名函数、lambda表达式、闭包closure

主流编程语言都对函数式编程有支持，比如c++11、python和java中有lambda表达式、lua和JavaScript中有闭包（closure）、ObjectC中有块（blocks，^）

lambda就是匿名函数，从功能性上说lambda和closure（或是OC中的blocks）是一个东西，只是不同语言的不同称呼罢了，它们都是匿名函数。若匿名函数捕获了一个外部变量，那么它就是一个closure。
捕获变量就是让匿名函数可以使用匿名函数外定义的变量，但是匿名函数内的函数外变量是外部变量的一个clone（C++11支持捕获引用），在匿名函数内部修改外部变量不会影响到外部变量。

C++中的lambda和python中的区别

在C++中lambda语法形式如下：
[capture](params)ops->ret{body;}
capture是捕获列表，params是参数表，opt是可选选项，ret是返回值类型，body是函数体。
可以捕获外部变量，还可以捕获引用。

python中lambda只能捕获外部变量，修改对外部变量没有影响。

lambda与def的区别

从名称上看，一个是匿名函数，一个是函数，所以两者的区别肯定有一条：有没有名字。实际上他们的区别主要有以下几条：

  • lambda是一个表达式expression，而def是一个语句statement
    lambda作为一个表达式与def相比，可以出现在更多的地方，比如在列表中，可以直接

  • lambda的函数体只是一个简单的表达式，不能有复杂的语句块
    lambda中没有return语句，也不能使用if、while、print等

    lambda is designed for coding simple functions, and def handles larger tasks.

  • lambda会返回一个函数对象，但这个对象不会赋给一个标识符，而def则会把函数对象赋值给一个变量（函数名）。
    万物皆对象，函数也是一个对象。

lambda生成了一个函数对象，该函数对象赋值给了func，func的调用和正常函数无异。

  • lambda函数不能共享给别的程序调用，def可以。

lambda使用

lambda语法如下lambda [arg1 [, agr2,.....argn]] : expression

lambda表达式会返回一个函数对象，如果没有变量接受这个返回值的话，它很快就会被丢弃。也正是由于lambda只是一个表达式，所以它可以直接作为list和dict的成员

在这里就没办法使用def语句替代了，语句是不能嵌套在里面的。

map、filter、reduce

map

map会将一个函数映射到一个输入列表的所有元素上。
函数原型为map(function, sequence)，对sequence中的item依次执行function(item)，将执行结果组成一个List返回

filter

filter过滤列表中的元素，并且返回一个由所有符合要求的元素所构成的列表。
函数原型为filter(function, sequence)，对sequence中的item依次执行function(item)，将执行结果为True的item组成一个List/String/Tuple（取决于sequence的类型）返回。

reduce

函数原型为reduce(function, sequence, starting_value)，对sequence中的item顺序迭代调用function，如果有starting_value，还可以作为初始值调用。
